# Sample Datasets

The `data/` directory is reserved for sample datasets that can be used to test AUDIT ALP's ingestion pipelines in the future.

Currently, the "Demo Mode" generates in-memory representations of the following documents:
- `Treasury_Transfer_Log_Q3.csv`
- `Treasury_RCM_v2.pdf`
- `Corporate_Policy_2026.pdf`
- `Previous_Audit_Q2.pdf`
- `Vendor_Master_Data.xlsx`
- `General_Ledger_Extract.csv`

*(Note: In a full production scenario with genuine OCR and Document Intelligence capabilities, real files placed in this directory would be parsed by the backend worker.)*
